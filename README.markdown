# lowryduda generator

This is a static site generator specialized for the creation of my professional
webpage. The pages are created from markdown files in `markdown/` according to a
certain schema of inclusion described below.


# Schema

This scheme defines four major pages and the markdown files they eat to produce
the final page.

- index.html
  * personal, contact info
  * travel info (if relevant)
  * education info
  * publication info
  * teaching info
  * programming
  * talk info
  * service
  * programming service

- publication page

- teaching page

- programming page
